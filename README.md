Company Management System
Overview
This system was developed to meet all the company's needs by streamlining the work of all departments. It simplifies tasks, enhances efficiency, and ensures secure access and management of data across the company. The system is built using PHP Data Objects (PDO), providing a reliable, secure, and scalable solution for managing databases.

Key Features
Department Management: Facilitates the organization and management of various company departments.
Task Automation: Automates routine tasks, reducing the manual workload for employees.
Secure Database Interaction: Developed with PDO, ensuring secure and efficient interactions with the company's database, preventing SQL injection attacks.
User-Friendly Interface: Easy to navigate and use, making it accessible for users across different departments.
Access Control: Manages user sessions and permissions, allowing only authorized personnel to access sensitive data.

Technologies Used
PHP (PDO): For secure and flexible interaction with the database.
MySQL: Database management.
WAMP (Windows, Apache, MySQL, PHP): Local development environment.

Installation
Clone the repository:
git clone https://github.com/saleck24/systeme-pdo.git
Start your local server (WAMP/XAMPP).
Import the database file (db.sql) located in the database folder into your MySQL.
Update the database connection details in the config.php file to match your local environment.
Open your browser and navigate to http://localhost/project1.
